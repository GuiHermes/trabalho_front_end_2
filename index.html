Trabalho Front-End 2

Este site foi desenvolvido para o trabalho da disciplina de Front-End 2, utilizando HTML, CSS e JavaScript. O objetivo é criar um formulário que permita ao usuário preencher seus dados, validar as informações inseridas e confirmar o envio de forma simples.

O sistema possui três páginas principais:

  • Início: mostra as informações do projeto
  • Formulário: onde o usuário preenche nome, sobrenome, e-mail e idade.
  • Confirmação: exibe os dados para revisão ou baixar em .json.
